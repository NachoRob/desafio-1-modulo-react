# 🍕 Pizzería Mamma Mia! - React Project

¡Bienvenido al frontend de **Mamma Mia!**, una aplicación reactiva construida con **React + Vite**. Este proyecto es parte del desafío de React de Desafío Latam, enfocado en el manejo de componentes, props y diseño responsivo con Bootstrap.

---

# Ver el Proyecto en Vivo:

[Visita mi proyecto aquí](https://desafio-1-modulo-react.vercel.app/)

---

## 🚀 Características (Features)
* **Diseño Responsivo:** Adaptado para móviles y escritorio usando **React-Bootstrap**.
* **Navegación Dinámica:** Navbar con renderizado condicional (Login/Register vs Profile/Logout).
* **Catálogo de Productos:** Visualización de pizzas mediante componentes reutilizables (`CardPizza`).
* **Formateo de Datos:** Uso de `toLocaleString()` para precios chilenos.
* **Header Impactante:** Hero section con diseño personalizado y filtros de imagen.

---

## 🛠️ Tecnologías Utilizadas
* **React:** Biblioteca principal para la interfaz.
* **Vite:** Herramienta de construcción (Bundler) ultra rápida.
* **Bootstrap 5:** Framework de CSS para el layout.
* **React-Bootstrap:** Componentes de Bootstrap listos para React.
* **Lucide React / Emojis:** Para la iconografía visual.

## 📦 Instalación y Uso

Si quieres probar este proyecto localmente, sigue estos pasos:

1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/TU_USUARIO/pizzeria-mamma-mia.git](https://github.com/TU_USUARIO/pizzeria-mamma-mia.git)
    ```
2.  **Entra a la carpeta:**
    ```bash
    cd pizzeria-mamma-mia
    ```
3.  **Instala las dependencias:**
    ```bash
    npm install
    ```
4.  **Corre el servidor de desarrollo:**
    ```bash
    npm run dev
    ```
    *Luego abre [http://localhost:5173](http://localhost:5173) en tu navegador.*
